Where does agent data go?

When you click ▶ Run on an agent in Overview:

📊 Funnel Agent → Reads conversations, builds funnel analytics → writes to daily_insights table

💡 Insights Agent → Analyses stage patterns, objections, conversion rate → writes to daily_insights

⚡ Follow-up Agent → Finds stale leads (no reply 24h+), sends automated follow-up → logs to messages table

🧬 Improve Agent → Reads patterns, generates improvement recommendations → writes to daily_insights

Reports appear in this tab after running. Check the database at Supabase → Table Editor → daily_insights for raw data.
